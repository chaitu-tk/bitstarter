Convertifi is a product idea, which enables us to convert any ethernet based device(legacy device) in to a WiFi Without the need for any changes to the legacy deice, its just plug and play. For this we make use traditional wifi technology, with the wifi device acting as a bridge/router in AP/STiA.

For Example, In the left image we see a camera with RJ45(ethernet) as the only network connection, now if we want to make this a Wi-Fi Enabled Device a normal USB Wifi Dongle will not work as there is no USB port and even if there is one, we do not have the flexibility to install the device drivers for that we dont have control the OS/Processor in the camera.

So we need some way where we can use the existing RJ45 port and conver it into a Wi-Fi enabled one, that what convertifi does.

Now imagine a touch screen capability to this to configure the "convertifi" without needing any PC, it is just awesome.

This is very useful to convert old but new enough to have a ethernet port camera, printers, fridges, TV's with an ethernet port to Wi-Fi, a niche market we want to captilize and serve.

AP Mode:

It can be used as a PocketWifi, where we can share the ethernet WAN connection to other Wifi users

STA Mode:

This is the primary selling point of this product, weherein we can connect this device to any legacy
One using the RJ45 slot and then connect the WiFi device to an Access Point using a touch screen. No need to connect to a PC for configuration.

FAQ's

How is it different from the traditional Access Point?
Traditional Access Point doesnt serve our purpose as it helps us to connect and share the internet to multiple device, here we just want to connect a legacy device with no wifi to a wifi network. Traditional AP is simply an underuse for this nice of usecases. Formfactor is another thing we need to think about.
How is it different from the traditional USB client Adapter?
Even though STA mode serves our purpose, but for an USB STA to wok we need to support from the legacy device (read device drivers) which isn't feasible.
